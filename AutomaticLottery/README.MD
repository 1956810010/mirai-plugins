### 插件介绍
针对群禁言抽奖插件，创建的插件自动化抽奖，获得禁言次数后可以禁言别人。
### 插件用法
后台使用命令，/alUser add qq号 添加权限；/alUser remove qq号 移除权限；/alUser list 显示用户列表，有用户权限后，使用“开始自动化”或者“启动自动化”开始自动抽奖，使用“停止自动化”或者“关闭自动化”关闭自动化抽奖，使用“捂住嘴巴”或者“闭嘴”加@需要禁言的人，禁言此人。
### 涉及指令
*/alUser*
