### 插件介绍
对服务器发送命令，可在QQ群内或私聊执行服务器命令。
### 插件用法
打开plugins/RCON/config.yml<br>修改serverAdder为Rcon服务器地址<br>serverPort为Rcon服务器端口<br>passworld为Rcon服务器端口<br>isPe填写: 如果是PE服务器填写true 如果是pc服务器填写false<br>注意！必须修改填写正确插件才加载！<br>如：<br>serverAdder: play.ltcraft.cn<br>serverPort: 19132<br>passworld: 你的Rcon密码<br>isPe: true<br>填写完成后在群内或私聊执行<br>/c 命令即可<br>如/c list
