### 插件介绍
群内看涩图, 可指定部分群不可看/只可看非R18/可看R18
### 插件用法
群内使用指令/hso normal允许群内发送普通涩图<br>群内使用指令/hso r18允许群内发送R18涩图<br>激发涩图的关键字可在配置文件中更改<br>默认为 色图->normal; 不够色->r18<br>请尊重所在国法律, 仅为学习所用
### 涉及指令
*/hso*
