### 插件介绍
支持群内看高清涩图, 允许配置每个群允许,不允许看普通/r18的图
### 插件用法
/hso noraml授权一个群可以看普通涩图<br>/hso r18授权一个群可以看r18涩图<br>触发图片的词为[色图/不够色], 可以在配置文件中更改
### 涉及指令
*/hso*
